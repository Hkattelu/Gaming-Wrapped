# Technology Stack: Gaming Wrapped

## Frontend
- **Framework:** Next.js 16 (App Router)
- **Library:** React 19
- **Styling:** Tailwind CSS, Tailwind CSS Animate
- **Components:** Radix UI Primitives (Accordion, Dialog, Tabs, etc.)
- **Icons:** Lucide React
- **Animations:** Framer Motion
- **Charts:** Recharts

## Backend & Infrastructure
- **Platform:** Firebase
- **Database:** Cloud Firestore
- **Serverless:** Firebase Admin SDK
- **Environment Management:** Dotenv

## Artificial Intelligence
- **Framework:** Genkit AI (@genkit-ai/google-genai, @genkit-ai/next)
- **Modeling/Validation:** Zod

## Languages & Core Utilities
- **Language:** TypeScript
- **CSV Parsing:** Papaparse
- **Date Utilities:** Date-fns
- **DOM Utilities:** Cheerio
- **Schema Validation:** Zod

## Development & Testing
- **Package Manager:** npm
- **Testing:** Jest, React Testing Library, ts-jest
- **Linting:** ESLint, Stylelint
- **Type Checking:** TypeScript (tsc)
- **Local Development:** Firebase Emulators, Genkit CLI, tsx, vibe-kanban-web-companion
